01. 변수 : 데이터 저장

자바스크립트에서 변수(variable)는 데이터 값에 이름을 지정하여 사용하고 싶을 때 사용하는 것으로 값 변경이 가능합니다.
변수 안에는 숫자, 문자, 배열, 객체, 함수 를 넣을 수 있습니다. 

{
            let x = 100;
            let y = 200;
            let z = "javascript";

            console.log(x);
            console.log(y);
            console.log(z);
}

- 변수 선언시 예전에는 var 키워드를 사용했지만, 현재는 let 키워드를 사용함.
* var / let / const
- var : 중복 선언 가능, 재할당 가능
- let : 중복 선언 불가능, 재할당 가능
- const : 중복 선언 불가능, 재할당 불가능

결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

let: 변수를 선언할 때 사용되며, 선언 후에 값의 재할당이 가능합니다. 즉, 변수의 값이 필요에 따라 변경될 수 있습니다.
- 블록 스코프 내에서 변수를 선언하고 값을 재할당 

{
            let x = 100;
            let y = 200;
            let z = "javascript";

            x = 200;
            y = 300;
            z = "jQuery";

            console.log(x);
            console.log(y);
            console.log(z);
}

처음에는 x에는 100, y에는 200, z에는 "javascript"가 할당되어 있음
그 후에는 각 변수에 새로운 값이 할당되어 x는 200, y는 300, z는 "jQery"로 변경됨.

결과 확인하기
200
300
jQuery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

'let'으로 선언된 변수는 추가 할당이 가능합니다. 

{
            let x = 100;
            let y = 200;
            let z = "javascript";

            x += 100;
            y -= '100';
            z += 'jQuery';

            console.log(x);
            console.log(y);
            console.log(z);
}

* 할당연산자
- 변수에 값을 할당하고 동시에 해당 변수의 값을 조작하는 데 사용
ex. x += 100 >>> "+=" 는 할당연산자로 "현재 변수의 값에 다른 값을 더하고 그 결과를 변수에 할당, (x = x + 100) 의미와 동일

결과 확인하기
200
100
javascriptjQuery

04. 상수 : 데이터저장 + 데이터변경(X)

const: 상수를 선언할 때 사용되며, 한 번 할당된 이후에는 값의 재할당이 불가능합니다. 즉, 상수는 한 번 할당되면 그 값을 변경할 수 없습니다.
- const를 사용하여 상수 x, y, z를 선언하고 초기값을 할당한 후에 값을 출력 

 {
            const x = 100;
            const y = 200;
            const z = "javascript";

            console.log(x);
            console.log(y);
            console.log(z);
}

- 상수는 프로그램 내에서 일정한 값을 나타내고자 할 때 유용함.

결과 확인하기
100
200

11. 배열 : 데이터 저장(여러개) : 표현방법 7

- 배열 펼침 연산자(...)를 사용하여 배열을 펼쳐서 개별 요소로 분리하는 연산 수행

{
        const arr1 = [100, 200, 'javascript']
        const arr2 = [300, 400, 'jquery']

        console.log(arr1[1]); // 1번
        console.log(arr1[2] + arr2[2]); // 2번
        console.log(arr1); // 3번
        console.log(...arr1) // 4번
        console.log(...arr1, ...arr2) // 5번
}

*스프레드 문법 (...)
- ... 연산자를 사용하여 배열이나 문자열을 개별 요소로 분해하여 결합할 수 있음.
ex. 주로 배열과 객체의 결합, 복사, 함수 호출 시 인수 전달 등 다양한 상황에서 사용됨.

결과 확인하기
1. 200
2. javascriptjquery
3. [100, 200, 'javascript']
4. 100 200 'javascript' (쉽표제거됨)
5. 100 200 'javascript' 300 400 'jquery'

22. 자료형(DataType)

원시 타입 (Primitive Type)

1. number(숫자) : 정수(integer)와 소수(floats)를 모두 포함합니다.
2. string(문자열) : 텍스트 데이터를 나타내며, 작은 따옴표(')나 큰 따옴표(")로 둘러싸여 있습니다.
3. boolean(논리형) : 참(True) 또는 거짓(False)을 나타내는 자료형입니다.
4. undefined : 값이 할당되지 않은 변수의 초기값을 나타냅니다. 변수를 선언했지만 값을 할당하지 않은 경우에 해당됩니다.
5. null : 값이 없음을 나타냅니다. 프로그램에서 명시적으로 '값이 없음'을 나타낼 때 사용됩니다.

{
        let integerNumber = 10;    // 1. 정수
        let floatNumber = 3.14;    // 1. 소수

        let greeting = "Hello, World!";   // 2. 문자열

        let isTrue = true;    // 3. 참
        let isFalse = false;  // 3. 거짓

        let notDefined;  // 4. 값이 할당되지 않은 변수
        let noValue = null;   // 5. 값이 없음       
}

참조 타입 (Reference Type)

1. 객체 (Object) : 여러 데이터를 하나로 묶어서 관리하는데 사용됩니다.

Object 특징
- 중괄호 {}로 둘러싸여 있습니다.
- 키(key)와 값(value)으로 이루어진 속성(property)들이 포함됩니다.
- 키는 문자열로 표현되며, 각 속성을 식별합니다.
- 값은 어떤 데이터 타입이든 될 수 있습니다. (숫자, 문자열, 배열, 함수 등)
- 데이터와 해당 데이터에 대한 동작(메서드)을 포함할 수 있습니다.
- 동적으로 속성을 추가, 수정, 삭제할 수 있습니다.
{                     
        let person = {
        name: "John Doe",
        age: 30,
        address: "123 Main St"
        }
        console.log("Name:", person.name);
        console.log("Age:", person.age);
        console.log("Address:", person.address);
}
결과 확인하기
Name: John Doe
Age: 30
Address: 123 Main St

2. 배열 (Array) : 여러 데이터를 순차적으로 저장하는 자료구조입니다.

Array 특징
- 대괄호 []로 둘러싸여 있습니다.
- 각 요소는 쉼표로 구분되며, 인덱스로 순서를 나타냅니다.
- 요소에는 어떤 데이터 타입이든 포함할 수 있습니다. (숫자, 문자열, 객체, 다른 배열 등)
- JavaScript 배열은 동적으로 크기를 조절할 수 있습니다.
- 배열 요소에 접근할 때는 인덱스를 사용합니다.
{                     
        let numbers = [1, 2, 3, 4, 5];
        let fruits = ["apple", "banana", "orange"];
        
        console.log(numbers[0]);
        console.log(fruits[1]);
    }
결과 확인하기
1
banana

23. 연산자

프로그래밍에서 값을 조작하고 계산하는 데 사용되는 기호 또는 기능입니다.
다양한 유형의 연산자가 있으며, 각각의 연산자는 특정한 작업을 수행합니다.

1. 산술 연산자 (Arithmetic Operators)

- 산술 연산자는 숫자 값을 사용하여 계산을 수행합니다.
• + : 덧셈
• - : 뺄셈
• * : 곱셈
• / : 나눗셈
• % : 나머지

[증가/감소 연산자]
• ++ : 증가 연산자. 변수의 값을 1씩 증가
• -- : 감소 연산자. 변수의 값을 1씩 감소 

{
        let a = 10;
        let b = 5;
        
        console.log(a + b); //결과: 15
        console.log(a - b); //결과: 5
        console.log(a * b); //결과: 50
        console.log(a / b); //결과: 2
        console.log(a % b); //결과: 0
        
        [증가/감소 연산자] ============================

        let x = 5;
        console.log(++x); // 출력: 6 (x가 증가된 후에 출력)
        console.log(x++); // 출력: 6 (x가 출력된 후에 증가)
        console.log(x);   // 출력: 7 (x의 값은 이전에 증가되어 6에서 7로 변함)

        let y = 10;

        console.log(--y); // 출력: 9 (y가 감소된 후에 출력)
        console.log(y--); // 출력: 9 (y가 출력된 후에 감소)
        console.log(y);   // 출력: 8 (y의 값은 이전에 감소되어 9에서 8로 변함)
}
2. 할당 연산자 (Assignment Operators)

- 할당 연산자는 변수에 값을 할당합니다.
• = : 변수에 값을 할당합니다.
• += : 현재 변수 값과 더하여 변수에 할당합니다.
• -= : 현재 변수 값에서 빼어 변수에 할당합니다.
• *= : 현재 변수 값과 곱하여 변수에 할당합니다.
• /= : 현재 변수 값을 나누어 변수에 할당합니다.
• %= : 현재 변수 값을 나눈 후 나머지를 변수에 할당합니다.

{
        let x = 10;
        x += 5; // x = x + 5;
        console.log(x); // 출력 결과: 15
        
        let y = 20;
        y -= 3; // y = y - 3;
        console.log(y); // 출력 결과: 17
        
        let z = 5;
        z *= 2; // z = z * 2;
        console.log(z); // 출력 결과: 10
}
3. 논리 연산자 (Logical Operators)

- 논리 연산자는 논리적 표현식을 평가하고 불리언 값을 반환합니다.
• && : AND 연산자
• || : OR 연산자
• ! : NOT 연산자

{
        let x = 10;
        let y = 5;
        
        console.log(x > 5 && y < 10); // 결과: true
        console.log(x > 5 || y > 10); // 결과: true
        console.log(!(x == 5)); // 결과: true
}
4. 비교 연산자 (Comparison Operators)

- 두 개의 값을 비교하고, 그 결과를 불리언 값으로 반환합니다. 주로 조건문이나 반복문에서 사용됩니다.
• == : 동등 연산자 (값 비교)
• === : 엄격한 동등 연산자 (값, Type 비교)
• != : 부등 연산자
• !== : 엄격한 부등 연산자 (값, Type 비교)

[관계연산자]
<: 작은지를 비교합니다.
>: 큰지를 비교합니다.
<=: 작거나 같은지를 비교합니다.
>=: 크거나 같은지를 비교합니다.

{
        let a = 5; // 변수 a에 5 할당
        let b = "5"; // 변수 b에 문자열 "5" 할당

        console.log(a == b); // true (형 변환 후 값이 같으므로)
        console.log(a === b); // false (타입이 다르므로)
        console.log(a != b); // false (형 변환 후 값이 같으므로)
        console.log(a !== b); // true (타입이 다르므로)
        console.log(a > b); // false (숫자와 문자열 비교 시, 문자열을 숫자로 변환하여 비교하므로)
        console.log(a < b); // false (숫자와 문자열 비교 시, 문자열을 숫자로 변환하여 비교하므로)
        console.log(a >= b); // true (숫자와 문자열 비교 시, 문자열을 숫자로 변환하여 비교하므로)
        console.log(a <= b); // true (숫자와 문자열 비교 시, 문자열을 숫자로 변환하여 비교하므로)
}
5. 삼항 연산자(Ternary Operator)

- 조건식을 평가하고 그 결과에 따라 두 가지 표현식 중 하나를 반환합니다.
[형식: condition ? expression1 : expression]
- condition : 평가할 조건식
- expression1, expression2 : 조건식이 참(true)일 때와 거짓(false)일 때 각각 반환될 표현식

{
        let num = 10;
        let result = (num % 2 === 0) ? "짝수" : "홀수";
        console.log(result); // 출력: "짝수" (10은 짝수이므로)
}
