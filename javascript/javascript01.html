01. 변수 : 데이터 저장

자바스크립트에서 변수(variable)는 데이터 값에 이름을 지정하여 사용하고 싶을 때 사용하는 것으로, 값 변경이 가능합니다.
변수 안에는 숫자, 문자, 배열, 객체, 함수 를 넣을수 있다. 

{
            let x = 100;
            let y = 200;
            let z = "javascript";

            x *= 100; // x = x + 200
            y -= '100';
            z += 'jQery';

            console.log(x);
            console.log(y);
            console.log(z);
}

- 변수 선언시 예전에는 var 키워드를 사용했지만, 현재는 let 키워드를 사용함.
* var / let / const
- var : 중복 선언 가능, 재할당 가능
- let : 중복 선언 불가능, 재할당 가능
- const : 중복 선언 불가능, 재할당 불가능

결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

let: 변수를 선언할 때 사용되며, 선언 후에 값의 재할당이 가능합니다. 즉, 변수의 값이 필요에 따라 변경될 수 있습니다. 

{
            let x = 100;
            let y = 200;
            let z = "javascript";

            x = 200;
            y = 300;
            z = "jQery";

            console.log(x);
            console.log(y);
            console.log(z);

}

결과 확인하기
200
300
jQery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

'let'으로 선언된 변수는 추가 할당이 가능합니다. 

{
            let x = 100;
            let y = 200;
            let z = "javascript";

            x += 100;
            y -= '100';
            z += 'jQery';

            console.log(x);
            console.log(y);
            console.log(z);

}

* 할당연산자
x += 100 >>> "+=" 는 할당연산자로 "현재 변수의 값에 다른 값을 더하고 그 결과를 변수에 할당
(x += 100) = (x = x + 100) 의미와 동일

결과 확인하기
200
100
javascriptjQery

04. 상수 : 데이터저장 + 데이터변경(X)

const: 상수를 선언할 때 사용되며, 한 번 할당된 이후에는 값의 재할당이 불가능합니다. 즉, 상수는 한 번 할당되면 그 값을 변경할 수 없습니다. 

 {
            const x = 100;
            const y = 200;
            const z = "javascript";

            console.log(x);
            console.log(y);
            console.log(z);

}

결과 확인하기
100
200
javascript

10. 배열 : 데이터 저장(여러개) : 표현방법 6

배열에서 원하는 값을 추출하여 변수에 할당할 때 유용합니다. 

{

- 선언 이후 값을 할당하지 않은 변수는 undefined값을 가짐

11. 배열 : 데이터 저장(여러개) : 표현방법 7

배열 펼침 연산자

{
        const arr1 = [100, 200, 'javascript']
        const arr2 = [300, 400, 'jquery']

        console.log(arr1[1]); // 1번
        console.log(arr1[2] + arr2[2]); // 2번
        console.log(arr1); // 3번
        console.log(...arr1) // 4번
        console.log(...arr1, ...arr2) // 5번
}

*스프레드 문법 (...)
- ... 연산자를 사용하여 배열이나 문자열을 개별 요소로 분해하여 결합할 수 있음.
ex. 주로 배열과 객체의 결합, 복사, 함수 호출 시 인수 전달 등 다양한 상황에서 사용됨.

결과 확인하기
1. 200
2. javascriptjquery
3. [100, 200, 'javascript']
4. 100 200 'javascript' (쉽표제거됨)
5. 100 200 'javascript' 300 400 'jquery'
