01. if문

조건문(if statement)은 프로그래밍에서 특정 조건이 참(true)인 경우에만 특정 코드 블록을 실행하는 데 사용됩니다. 

{
        if (true) {
            document.write("실행되었습니다. (true)");
        } else {
            document.write("실행되었습니다. (false)")
        }
}

[true로 평가되는 값]
1. true: 불리언(true) 값으로, 직접적으로 참을 나타냄
2. "문자열": 비어있지 않은 문자열은 항상 true로 평가
3. 1: 숫자 1은 true로 평가

[false로 평가되는 값]
1. false: 불리언(false) 값으로, 직접적으로 거짓을 나타냄
2. 0: 숫자 0은 false로 평가
3. null: null은 명시적으로 '값이 없음'을 나타내며, false로 평가.
4. undefined: 변수에 값이 할당되지 않았을 때의 상태를 나타내며, false로 평가
5. "" (빈 문자열): 비어있는 문자열은 false로 평가

02. 다중 if문

여러 개의 조건을 검사하여 각각에 맞는 코드 블록을 실행합니다.
각 if문은 순차적으로 평가되며, 조건이 참일 때 해당하는 코드 블록이 실행됩니다.

{
        const num = 100;

        if (num == 90) {
            document.write("실행되었습니다. (num = 90)");
        } else if (num == 95) {
            document.write("실행되었습니다. (num = 95)");
        } else if (num == 100) {
            document.write("실행되었습니다. (num = 100)");
        } else if (num == 105) {
            document.write("실행되었습니다. (num = 105)");
        };
}

결과 확인하기
실행되었습니다. (num = 105)

03. 중첩 if문

중첩된 if문은 하나의 if문 안에 또 다른 if문이 포함되어 있는 구조입니다.
내부의 if문은 외부의 if문의 조건이 참일 때만 실행됩니다.
특정 조건이 만족되었을 때 추가적인 조건을 검사하고자 할 때 유용하게 사용됩니다. 

{
        const num = 100;

        if (num == 100) {
            document.write("실행되었습니다. 1");
            if (num === 100) {
                document.write("실행되었습니다. 2");
            } if (num === 100) {
                document.write("실행되었습니다. 3")
            }
        } else {
            document.write("실행되었습니다.")
        }
}

*주의해야 할 점은 중첩된 if문의 조건은 외부의 if문의 조건이 참인 경우에만 검사됩니다.
따라서 외부 if문의 조건이 거짓이면 내부 if문은 검사조차 되지 않습니다.

결과 확인하기

04. if문 생략 & 삼항 연산자

1. if문 생략 (한 줄 if문)
- if문의 실행문이 하나뿐이기 때문에 중괄호를 생략할 수 있습니다.
2. 삼항연산자 (형식 : condition ? expression1 : expression2)
- condition은 평가될 조건식입니다.
- expression1은 조건식이 참일 경우 반환될 표현식입니다.
- expression2는 조건식이 거짓일 경우 반환될 표현식입니다. 

{
        const num = 100;
        // 1. if문
        if (num == 100) {
            document.write("실행되었습니다. (true)");
        } else {
            document.write("실행되었습니다. (false)");
        }
        
        // 2. if문 생략
        if (num == 100) document.write("실행되었습니다. (true)");
        else document.write("실행되었습니다. (false)");
        
        // 3. 삼항연산자
        (num == 100) ? document.write("true") : document.write("false");
}

결과 확인하기
1. 실행되었습니다. (true)
2. 실행되었습니다. (true)
3. true

05. switch문

여러 경우(case)에 따라 다른 동작을 수행해야 할 때 사용됩니다.
주어진 표현식의 값을 기준으로 여러 개의 case를 비교하고, 일치하는 case가 발견되면 해당 case의 코드 블록을 실행합니다.

{
        const num = 100;

        switch (num) {
            case 90:
                document.write("실행되었습니다.(90)");
                break;
            case 100:
                document.write("실행되었습니다.(100)");
                break;
            case 105:
                document.write("실행되었습니다.(105)");
                break;
            case 110:
                document.write("실행되었습니다.(110)");
                break;
            default:
                document.write("실행되었습니다.");
        };
}

*default 절
switch문에는 모든 case와 일치하지 않을 때 실행될 default 절을 포함할 수 있습니다.
이를 통해 예기치 않은 상황에 대한 대비책을 마련할 수 있습니다.

결과 확인하기
실행되었습니다.(110)

06. while문

while문은 조건이 참인 동안 반복적으로 코드 블록을 실행하는 반복문입니다.
조건이 참이면 코드 블록이 실행되고, 다시 조건을 검사하여 조건이 여전히 참이면 코드 블록이 반복해서 실행됩니다.
이 과정은 조건이 거짓이 될 때까지 계속됩니다. 

{
        let num = 1;
        while (num <= 10) {
            document.write(num + "
")
            num++;
        }
}

08. for문

for문은 프로그래밍에서 반복 작업을 수행하는 데에 사용되는 제어 구조 중 하나입니다.
일정한 횟수만큼 반복을 수행하거나, 배열과 같은 데이터 구조의 각 요소에 대해 반복 작업을 수행하는 데에 매우 유용합니다.

{
        for (let i = 0; i <= 10; i++) {
            console.log(i)
        }
}

*for문의 실행순서
1. 초기식 실행
2. 조건식이 평가 (참이면 반복문이 실행되고, 거짓이면 종료)
3. 반복문이 실행
4. 증감식이 실행
5. 다시 조건식 평가됨을 거짓이 될때까지 반복

09. break문

break문은 반복문이나 switch문에서 실행을 즉시 중지하는 데 사용됩니다.

{
        for (let i = 1; i <= 10; i++) {
            if (i == 5) {
                break;
            }
            document.write(i, "
");
        }
}

1. 반복문에서의 break
- 반복문 안에서 break가 실행되면 해당 반복문이 즉시 종료됩니다.
- 이후의 코드 블록은 실행되지 않고 반복문을 빠져나옵니다.
- 주로 반복문 내에서 특정 조건을 만족하면 반복문을 종료할 때 사용됩니다.
2. switch문에서의 break
- break는 switch문에서 각 case 블록의 실행을 중지하고 switch문을 빠져나오는데 사용됩니다.
- break문이 없으면 case 블록이 실행된 이후에도 계속해서 다음 case 블록이 실행됩니다.

10. continue문

continue문은 현재 반복을 중지하고 다음 반복으로 넘어가는데 사용됩니다.

{
        for (let i = 0; i < 10; i++) {
            if (i == 5) {
                continue;
            }
            document.write(i, "
");
        }
}
