01. 변수 : 데이터 불러오기

{
        let x = 100, y = 200, z = "javascript"; //이어쓰기 가능
        console.log(x, y, z); // 100 200 'javascript'  
}

*이어쓰기(Chaining)
- 여러 변수를 한 줄에 선언하고 초기화할 때 쉼표(,)로 변수를 구분합니다.

결과 확인하기
100
200
javascript

05. 배열 : 데이터 불러오기 : for()

- for문은 일반적인 반복 작업에 사용됩니다.
- 초기화 구문, 조건식, 반복 후 실행되는 구문이 모두 명시적으로 제어됩니다.
- 보통 반복 횟수가 명확할 때 사용됩니다.

{
        const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

        for (let i = 0; i < arr.length; i++) {
            console.log(arr[i]);
        };
}

*사용 상황
for : 반복 횟수가 명확하고 특정 범위 내에서 반복하는 경우 사용
forEach : 배열 또는 컬렉션을 순회하면서 요소를 접근해야 할 때는

결과 확인하기

200
300

700

06. 배열 : 데이터 불러오기 : forEach()

- foreach문은 배열 또는 컬렉션 같은 데이터 구조를 순회할 때 사용됩니다.
- 컬렉션의 요소들을 하나씩 가져와서 처리할 때 편리합니다.


        /* forEach() 형식 */

        array.forEach(function(element, index, array) {
            (각 요소에 대해 수행할 작업)
        });

        // ex.
        const num = [100, 200, 300, 400, 500];
        num.forEach(function (element, index, array) {
            console.log(element) // 첫번째인자 : 값
            console.log(index) // 두번째인자 : 인덱스
            console.log(i3) // 세번째인자 : 전체 배열
        });
*각 매개변수의 역할 (element, index, array)
1. element : 현재 처리 중인 배열 요소의 값
2. index : 현재 처리 중인 배열 요소의 인덱스
3. array : forEach()를 호출한 배열 자체

{
        const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

        arr.forEach(function (element) {
            console.log(element);
        });
}
결과 확인하기
100
200
300

600
700

900
